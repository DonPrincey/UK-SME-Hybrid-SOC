# Incident Investigations

This folder contains sanitised and simulated security incident investigations for the UK SME Hybrid SOC project.

Each investigation should document:

- Incident title and reference number
- Incident summary
- Initial alert
- Severity and priority
- Investigation timeline
- Evidence examined
- Queries used
- Indicators of compromise
- Affected users, hosts or systems
- MITRE ATT&CK techniques
- Containment actions
- Remediation recommendations
- Lessons learned
